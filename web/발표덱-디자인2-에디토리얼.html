Executive Summary

한 장 요약

받은 문제
"마케팅 성과 데이터를 분석해 비즈니스 인사이트 리포트를 만들어라." 5개 채널 8주치 일별 성과가 CSV로 제공됨.
만든 것
성과표를 넣으면 채널별 효율 순위, 이번 주 놓치면 안 될 이슈, 다음 주 예산 재배분안까지 한 번에 나오는 리포트.
핵심 임팩트
반나절 → 1분
리포트 작성 시간. 그리고 새고 있던 광고비 319만원을 찾아 옮기면 기대 추가매출 약 1,876만원.
결과물
리포트(md/HTML) · 재현 스크립트 · 계산값 원본 · 의사결정 로그
Problem Definition

문제 이해

기업이 낸 문제
주어진 성과 CSV로 "비즈니스 인사이트 리포트" 작성 요구. 포함할 인사이트 항목은 미지정.
재정의한 진짜 문제
담당자가 매주 답해야 할 질문은 하나. "다음 주 예산을 어디로 옮기나". 해당 답이 성과표에 정리되지 않은 것이 문제.
지금은 이렇게 (Before)
매주 5개 채널을 수작업 계산·요약, 반나절 소요. 엑셀 수식은 오류 검증 불가, 결과물은 수치 나열로 의사결정 미연결.
푼 것 / 안 푼 것
푼 것: 채널 효율 진단, 이슈 탐지, 예산 재배분안.
안 푼 것: 광고 플랫폼 자동 연동, 채널 간 기여도 정밀 분석은 이번 범위 밖.
Approach & Decision

해결 접근 & 의사결정

접근 가설
계산은 기계에 위임해 오류를 차단하고, "왜 문제인가·무엇을 할 것인가" 판단만 분석자가 수행. 매주 동일 품질 반복 확보.
핵심 흐름 (동작 기준)
입력
성과표 넣기

→
결과
순위·이슈·재배분안
버린 대안 & 이유
엑셀 수동 계산: 재현 불가·오류 미검출로 미채택. 판단의 기계 위임: 근거 설명 불가로 미채택.
★ 막혔던 지점 & 돌파
이메일 매출 1건이 평균의 9.6배. 미처리 시 이메일 순위 과대·전체 예산 판단 왜곡. 해당 건을 제외하고 제외 사유까지 명시해 신뢰성 확보. 데이터 신뢰성이 결론의 전제임을 확인.
기계 vs 나의 판단
합계·비율·변화율 계산은 기계 전담. 이상치 기준·이슈 우선순위·예산 이동 방향은 분석자 판단.
Output & Proof

무슨 일을 대신해 주나
담당자의 반나절 수작업을 대체. 성과표 입력 시, 채널별 효율 순위 산정, 당주 손실 지점 식별, 익주 이동 예산 금액까지 제시.

Applicability & Reflection

적용 가능성 & 아쉬움 & 비전

우리 환경에 쓰려면
·
기존 성과표 컬럼 정합 시 그대로 구동.
·
업계 비교 기준값을 사내 데이터로 교체 시 정확도 향상.
솔직한 아쉬움
·
채널 간 기여도 중복 미분리. 이동 효과는 상한이 아닌 기준값 적용.
·
업계 비교 수치는 외부 자료로, 사내값 교체 필요.
2주 더 준다면
→
이동 후 실매출 자동 추적, 추정치와 대조.
→
이상 감지 시 리포트 자동 알림.
→
채널 기여도 분석 추가로 이동 효과 정밀화.
Closing

정식으로 맡으면 매주 예산 회의를 리포트 한 장으로 대체하고, 이동 효과까지 추적하는 구조로 키울 수 있습니다.

오늘 결과물이 근거입니다. 숫자를 정리하는 데서 멈추지 않고, 다음 주 무엇을 할지까지 답을 냈습니다.

Appendix

별첨. 기술 & 계산 상세

기술 검증을 원하는 분만. 본문은 비즈니스 언어로만 썼습니다.
계산 / 해석 분리
합계·ROI·변화율·이상치 탐지는 전부 Python(pandas)이 계산. LLM(Claude)은 계산 결과 해석과 권고만. 계산 오류 위험을 코드가 차단.
재현
python generate_insight_report.py --csv ... 한 줄로 리포트와 계산값 재생성. 실제 재실행해 수치 일치 확인.
정제 규칙
완전 중복 1행 제거 → 결측 3건은 0 대체 없이 해당 지표에서만 제외 → 이메일 이상치 1건 제외(원본 로그 확인 전 잠정) → 오가닉은 광고비 0이라 ROI 측정 불가로 분리.
