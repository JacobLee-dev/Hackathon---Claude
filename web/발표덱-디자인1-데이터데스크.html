PROBLEM DEFINITION

문제 이해

기업 원문 문제

5개 채널 성과 데이터 기반 마케팅 인사이트 리포트 작성 요구. 포함 항목은 미지정.

재정의한 문제

핵심 페인포인트는 리포트 부재가 아니라, 데이터는 있으나 '다음 주 예산 배분' 답이 도출되지 않는 점. 리포트가 아닌 예산 결정 도구로 재정의.

문제의 크기 (Before)

매주 5개 채널 성과를 수작업 추출·계산·요약, 반나절 소요. 엑셀 수식은 오류 검증 불가, 결과물은 수치 나열로 의사결정 미연결.

스코프 · 푼 것 / 안 푼 것

푼 것 채널 ROI·이슈·재배분안·재현 파이프라인.   안 푼 것 크로스채널 어트리뷰션, 실시간 연동, 자동 발송. 데이터 범위 밖으로 의도적 제외.
